## 1. 주요 구현 기능 (Key Features)

### 📱 [NEW] 지능형 모바일 UI/UX 최적화 (4/15)
- **반응형 바텀 시트(Bottom Sheet) 모달**:
    - `BranchDetailModal.js` 및 `RealtimeLockerList.js`를 모바일 화면에 최적화된 바텀 시트 방식으로 전면 개편.
    - 모바일 기기별 뷰포트 높이 문제를 해결하여 상하단 콘텐츠가 잘리는 현상을 완벽 차단 (`max-h-[92vh]` 적용).
- **스마트 내비게이션 고도화**:
    - **슬라이드 자동 감지**: 모바일 메뉴 열림 시, 화면을 스크롤(슬라이드)하거나 배경을 터치하면 메뉴가 자연스럽게 닫히는 로직 구현.
    - **로고 홈 리턴**: 어느 페이지에서든 로고를 누르면 홈 화면 최상단으로 부드럽게 스크롤되며 이동하는 동선 최적화.
- **모바일 퀵 예약 버튼 (FAB)**:
    - 우측 하단에 상시 고정된 '예약하기' 플로팅 버튼 추가.
    - **글래스모피즘 말풍선 가이드**: "지금 바로 예약하기"라는 안내 문구가 둥둥 떠 있는 애니메이션 효과를 더해 터치율과 편의성 극대화.

### 🔥 메인 랜딩 페이지 프리미엄화
- **시설 커스텀 캐러셀 (FeatureCards)**: 라이프스타일 사진 5장을 활용한 가로 스크롤 캐러셀 도입 및 Webkit 렌더링 버그 해결.
- **유튜브 쇼츠 갤러리 (ShortsGallery)**: 9:16 비율의 숏폼 썸네일을 활용한 생생한 이용 후기 섹션 구축.
- **랜딩 화면 로딩 최적화**: 이용후기(`/reviews`)와 자주 묻는 질문(`/faq`)을 별도 페이지로 분리하여 메인 페이지 성능 향상.

### 🚀 통합 예약 및 결제 프로세스
- **3단계 결제 플로우 (RealtimeLockerList)**: 지점 상태 확인 → 보관함 사이즈 선택 → 기간 설정 및 결제 단계를 일원화.
- **자유로운 기간 설정 (일할 계산)**: 월 단위 고정에서 벗어나 시작/종료일을 자유롭게 지정하고 실시간으로 요금을 계산하는 엔진 탑재.
- **멤버십 0원 결제**: 구독 회원은 추가 결제 없이 즉시 예약이 완료되는 프리패스 로직 연동.

### 🔐 시스템 안정성 및 마이페이지
- **하이드레이션(Hydration) 에러 제로**: 서버와 클라이언트 간의 렌더링 불일치(푸터 정보 등)를 `suppressHydrationWarning`으로 완벽 해결.
- **Supabase 최적화**: 무한 렌더링 루프를 방지하기 위해 클라이언트 인스턴스 생성을 컴포넌트 외부로 분리하고 안정성 확보.
- **지능적 소셜 로그인 대응**: 카카오/네이버 연동 유저에게는 비밀번호 변경 대신 소셜 연동 상태창을 노출하여 사용자 혼란 방지.

## 3. 핵심 프롬프트 가이드 (Prompt Guide)
- **모바일 UI 수정 시**: "모바일은 항상 바텀 시트(Bottom Sheet) 접근 방식을 유지하고, `max-height`를 설정하여 브라우저 상하단 바에 콘텐츠가 가려지지 않게 해줘."
- **서버 데이터 연동 시**: "Supabase 클라이언트는 전역 선언된 것을 재사용하고, 사용자의 `app_metadata.provider`를 통해 소셜 로그인 여부를 판단하여 처리해줘."
- **인터랙션 구현 시**: "사용자의 행동(스크롤, 외부 터치)에 반응하여 메뉴나 모달이 자연스럽게 닫히도록 이벤트 리스너를 적절히 배치해줘."
